如今我已是初二，回眸望去在中雅校区度过的一年，百感交集。2020年差不多10月的时候(初一上学期)，便参加了一次考试，是进入信息竞赛班和数学竞赛班的。奈何我数学不好，我也便没考上。一个星期后草泽却跟信息老师推荐，把我推荐进了信息班。这么说来，我和信息竞赛也挺有缘分的…

五年级开始钻研各种电脑上的东西，在上初中之前学Java也学到了SpringMMC。对于Java这么早学到这种程度的估计除了我也没多少人。但是学程序设计和web也对主要为算法竞赛的信息竞赛来说，也没什么帮助，所以我在班上也一直和别的同学差不多，并没有显得特别突出。

在初一上学期的时候我的成绩也不是很好，所以当时班主任也和我说过，“你说你成绩也不是很好，就别学这个信息了吧。”这个时候当然就是两个选择，一个是把成绩搞好，一个是不学信息。但是，我硬是撑着，成绩也没上去，信息也在学，就坚持了下来。当时我们班有五个人在信息班，初一期中考试之后就走了两个，只剩下三个人。

说实话，我对待信息课认真起来也就是这次比赛之后。可能也是因为换了个老师吧…以前上课除了讲新课我一般都不听或者不认真听，实在因为是太无聊了而不是我懒之类的。比完赛之后，信息A班和B班合并，对于B班的同学们来说换了个新老师，我就觉得这个老师讲课让人没那么困了，也决心好好听课。反正我待在这个学校还不是三年，过了一年了，拿了个二等奖。等下一次比赛，一定要拿一个提高组一等奖，不在中雅留下遗憾！

接下来才是正题：

初赛篇

嗯嗯，坐上校车到了湖南大学，惊奇的发现原来湖南大学没有校门。由于夏季风较强，到了十月份长沙还是三十六七度。虽然是早上，但是太阳晒着刚剪完头发的我，我还是感到阵阵火辣。漫长的排队过后，终于进考场了。一个考场三十个人左右吧，我认识的人在这个考场里面只有两个人。监考还是挺严的，每一次进出考场，都要重新检查一次有没有带别的东西。

等了半个多小时候，发了试卷。一看题目，大多数都是复习过的，但是仍然还是大部分不会。认认真真做了一个多小时的卷子，认认真真检查了半个小时(却已无力回天)，终究还是按耐不住了，提前了十五分钟交卷，离开考场。老师不让提前交卷怎么办呢？在楼梯间玩♂耍！在下楼梯时，遇到了一个班的同学，看他在刷B站，于是一起颓废了十分钟之后再出了楼。

之后等了好久好久好久好久好久好久好久好久好久好久…人终于齐了，坐了校车回家。

初赛也没什么好说的，算是给我这个第一次参加比赛的蒟蒻开了开世面。

最终成绩：48.5 最终评定：二等奖

复赛篇

在考试之前上了很多很多节课，每节课都是模拟考-讲题，模拟考-讲题。因为对我而言，模拟考的难度很高，我最高的一次也才170分，所以直接让我怀疑人生了——可能我真的不适合学信息，好好学文化，把你的数学拉上来，也不至于次次考试连年级前一百都没有。

在考试的前一天，突然老实说湖南地区由于疫情取消比赛了，给我当时开心得…于是那天中午就没有再复习，颓废了一中午。结果第二天，老师又说要考试，真是给我们来了个回马枪。考就靠呗，反正我也不是没有准备，我也得证明一下我学了这一年的信息不是白学的。

于是第二天，我又刷了很多题。但是用处不大，因为我没学懂的东西实在是太多了，我也只就刷了刷背包…所以在考试之前我一直都是很迷茫的状态——上课不怎么认真，平时不怎么做作业，遇到不会做的题网上一搜了之。

考试流程（注意事项文字版）：

  1. 湖南大学工商管理学院前坪集合

  2. 跟着老师指导进入考场

  3. 全身确认未带违规物品

  4. 打开电脑，进入Windows系统

  5. 打开电脑D盘，新建文件夹为准考证号

  6. 确保文件夹名准确，大、-、半角、数字位数

  7. 打开Dev-C++，提前写好缺省源、freopen

  8. 看老师黑板上的试卷下载地址，下载试卷

  9. 根据老师指示，通过密码打开试卷

  10. 打开试卷第1页，看到“目录”是不是子文件夹，是就建子文件夹，名字不要写错，所有文件对应题目命名

  11. 开始做题，1题送分，2题枚举或模拟或贪心，3题可能DP或搜索，4题可能涉及数据结构但是一定都是基本语句能解决的，一般难度为1<2<4<3

  12. 下考前5-10分钟，停止做题，检查所有的文件名、freopen、注释。确认在D盘保存好，等待下考。

  13. 下考，起身离开，不关电脑。

  14. 带着自信和微笑走到考点门口，拥抱熊老师

PS：有任何问题都找老师（除了题目）。

int nums[8010] = {0};
int nums2[8010] = {0};

// copy num1 + 1 ~ num1 + 1 + n to num2 + 1 ~ num2 + 1 + n
void copynum(int* num1, int* num2, int n) {
    for(int i = 1; i <= n; i++) {
        num2[i] = num1[i];
    }
}

int main() {
    int i = 0;
    freopen("sort.in", "r", stdin);
    freopen("sort.out", "w", stdout);
    int n, q;
    int a, x, v;
    scanf("%d%d", &n, &q);
    for(int i = 1; i <= n; i++) {
        scanf("%d", &nums[i]);
    }
    for(int i = 0; i < q; i++) {
        scanf("%d", &a);
        if(a == 1) {
            scanf("%d%d", &x, &v);
            nums[x] = v;
        } else if(a == 2) {
            copynum(nums, nums2, n);
            scanf("%d", &x);
            int temp = nums[x];
            int index = 0;
            for(int i = 1; i <= n; i++) {
                if(nums[i] == temp) {
                    index++;
                    if(x == i) {
                        break;
                    }
                }
            }
            sort(nums2 + 1, nums2 + 1 + n);
            for(int i = 1; i <= n; i++) {
                if(nums2[i] == temp) {
                    if(!(--index)) {
                        printf("%d\n", i);
                    }
                }
            }
        }
    }
    return 0;
}

bool iphf(string s) {
    bool isH = true;
    int tempnum = 0; // sum
    int _10c = 1; // 10 ^ _2c
    int index = 1;
    //123.456.789.123:45678
    if(s.length() < 9 || s.length() > 21) return false;
    if(s[s.length() - 1] < '0' || s[s.length() - 1] > '9') {
        return false;
    }
    for(int i = s.length() - 1; i >= 0; i--) {
        if((s[i] != ':' || s[i] != '.') && s[i] >= '0' && s[i] <= '9') {
            tempnum += (s[i] - 48) * _10c;
            _10c *= 10;
            if(!isH) {
                if(tempnum > 255) {
                    return false;
                }
            }
        } else if(s[i] == ':') {
            index++;
            if(index > 5) return false;
            if(i == 0 || i == s.length() - 1) {
                return false;
            }
            if(!isH) return false;
            if(tempnum > 65535) {
                return false;
            }
            _10c = 1;
            isH = false;
            tempnum = 0;
        } else if(s[i] == '.') {
            index++;
            if(index > 5) return false;
            if(i == 0 || i == s.length() - 1) {
                return false;
            }
            if(isH) return false;
            if(tempnum > 255) {
                return false;
            }
            _10c = 1;
            tempnum = 0;
        } else if(i == 0) {
            if(tempnum > 255) {
                return false;
            }
            _10c = 1;
            tempnum = 0;
        }
    }
    if(index != 5) {
        return false;
    }
    return true;
}

bool strIsame(string str1, string str2) {
    int len = str2.length();
    if(len != str1.length()) return false;
    for(int i = 0; i < len - 1; i++) {
        if(str1[i] != str2[i]) return false;
    }
    return true;
}

struct Com {
    string type; //Server or Client
    string ip;
    int number;

int main() {
    freopen("network.in", "r", stdin);
    freopen("network.out", "w", stdout);
    int n;
    cin >> n;
    for(int i = 0; i < n; i++) {
        cin >> coms[i].type >> coms[i].ip;
        coms[i].number = i + 1;
        if(!(iphf(coms[i].ip))) {
            cout << "ERR" << endl;
        } else if(strIsame(coms[i].type, "Server")) {
            int isok = true;
            for(int j = i - 1; j >= 0; j--) {
                if(strIsame(coms[j].type, "Server") && strIsame(coms[i].ip, coms[j].ip)) {
                    cout << "FAIL\n";
                    isok = false;
                    break;
                }
            }
            if(isok) {
                cout << "OK\n";
            }
        } else if(strIsame(coms[i].type, "Client")) {
            int isfati = true;
            for(int j = 0; j < n; j++) {
                if(strIsame(coms[j].ip, coms[i].ip) && strIsame(coms[j].type, "Server")) {
                    cout << coms[j].number << endl;
                    isfati = false;
                    break;
                }
            }
            if(isfati) {
                cout << "FAIL\n";
            }
        }
    }
    return 0;
}
