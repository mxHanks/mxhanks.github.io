由于静态网站的某些功能有限，所以我们需要第三方服务来扩展我们的网站。在任何时候，你都可以使用 NexT 支持的第三方服务来扩展所需的功能。

Next 提供了两个渲染引擎来显示数学方程: MathJax 和 KaTeX。

要使用这个特性，您只需要选择一个渲染引擎并打开它的 enable(位于主题的 _config.yml 。 然后你需要安装相应的 Hexo 渲染器来完全支持数学方程式的显示-只开启启用可能不会让你正确地看到显示的方程式。 相应的 Hexo 渲染引擎将提供如下：

开启之后使用per_page控制是否每页呈现数学方程式。

Render Engines 渲染引擎

目前，NexT 提供了两个渲染引擎: MathJax 和 KaTeX。

MathJax 引擎(推荐)

首先要确定你安装过了pandoc (版本 >= 2.0)

在主题设置文件中开启 mathjax:

KaTeX 引擎(暂不推荐)

目前在 NexT 主题中 KaTeX 还不完善, 暂时不建议使用.

与 MathJax 相比，KaTeX 引擎是一个更快的数学渲染引擎，而且没有 JavaScript 它也能生存。

需要卸载原始渲染器 hexo-renderer

如果你使用 KaTeX 渲染数学公式，你需要安装渲染器选中的一个:

在主题配置文件中，选择 katex 作为渲染引擎。

本地安装 pandoc

进入 pandoc安装页面，点击下载按钮并安装。

安装一定要选择 Install for all users! 否则在除了C:\Users\…\ 以外的目录都无法使用 pandoc 指令！(如果这样还不能解决就自己配置一下环境变量吧。。)

在 cmd 中输入 pandoc --version，如果成功显示版本即是安装成功。

**安装成功之后要重启电脑再 Hexo g ，否则将报错pandoc exited with code null。
